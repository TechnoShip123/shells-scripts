# Package Updater Script

### Dependencies
- Figlet (Not vital)
- Advanced Package Tool/APT (Vital)

### Info
This script is quite basic, it runs some standard apt commands to get package updates, upgrade all packages, and autoclean any packages that are no longer needed. You may need to enter your password since it uses sudo (apt requires elevated permissions).

### Breakdown
- Get package updates
- Update/Upgrade packages
- Autoclean/remove packages that are no longer required.
